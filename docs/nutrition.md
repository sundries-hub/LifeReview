# 营养品

a.k.a. 保健品



## Morelabs Aqua+ 电解质补充剂

喝了之后没有可见的效果。

作为一个经常口干的人，喝了这个之后感到口干的间隔基本上没有区别，但我也不知道我的身体是否「已经补充了水分」。这个看起来也不是那种针对运动后的补水，总之就是没有可见的效果。

口感还不错，加进纯净水里相比纯净水来说更容易喝的下去

买它的原因之一是盒子很好看，对于包装丑的东西我可能没有坚持服用的毅力...
